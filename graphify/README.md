# Graphify Output

This folder contains visualizations of the project's structure generated by **graphify**.

## How to generate the graph
```bash
# From the project root
graphify . -o graphify/graph.svg
```

The command will create an SVG file (`graph.svg`) that visualizes modules, imports, and dependencies.

You can open the SVG in any browser or image viewer.

## Updating the graph
Whenever you add or remove files, run the command again to refresh the visualization.
